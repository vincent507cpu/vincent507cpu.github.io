[工作站] 2020 年 10 月的多 GPU 深度学习工作站配置指南

发表于 2020-10-10

本文接上一篇《2020 年 10 月的单 GPU 深度学习工作站配置指南》，探讨多 GPU 工作站的搭建。很多在单 GPU 工作站中不甚重要的因素在多 GPU 工作站中变得举足轻重。

本文主要参考了以下文章：

  • Which GPU(s) to Get for Deep Learning: My Experience and Advice for Using GPUs in Deep Learning
  • A Full Hardware Guide to Deep Learning
  • Deep Learning Hardware Deep Dive – RTX 3090, RTX 3080, and RTX 3070
  • What’s the Difference Between a Blower and an Open-Air GPU Cooler?

    供电

工作站不能让电线过载，也不能超过电源的最大功率。对于中国用户来说，电线的最大功率不能超过入户线功率；对于美国用户来说，家庭/办公室的电线的峰值电流为 15A，电压为 120V。美国国家标准推荐最大使用功率不超过峰值功率的 80%，即 15 x 120 x 80% = 1440W。而常见的 CPU 与 GPU 的功率为：

CPU 功率
Threadripper 280W
Ryzen 7/9 105W
Core X-Series 165W
GPU 功率
RTX 3090 350W
RTX 3080 320W
RTX 3070 220W
RTX 2080 Ti 250W
Titan RTX 280W

目前市面上电源的最大功率为 1600W。考虑到主板还需要近 100W，就算使用功率最小的 Ryzen，同时使用 4 张 RTX 3080 显卡的功耗为 320 x 4 + 65 + 100 = 1445W，已经达到了推荐家用最大功率，同时电源几乎没有冗余。

综上，在不考虑散热的情况下，推荐的最多 GPU 数量为：3 张 3090/3080 或 4 张 3070。

散热

显卡的散热方式有风冷和水冷两种，风冷又分涡轮式散热（blower）和开放式两种（open-air）两种。

  • 开放式散热：由风扇吸入冷空气，冷空气在散热片上进行热交换，热空气在 GPU 的周围排出。

  • 涡轮式散热：整个 PCB 板被包裹起来，冷空气被风扇吸入后在散热片上进行热交换后在 GPU 后挡板处排出。

  • 水冷散热：冷水被水泵抽到芯片上吸收芯片的热量，热水随后被抽到散热片与冷空气进行热交换。


使用开放式散热的显卡会面临热空气被其它显卡吸收的问题，会降低散热的效果，极端情况下会造成显卡过热自动降频，从而降低性能。如果显卡之间有超过 1 个 PCI-E 空位，则基本不会存在散热的问题，但是这样由于空间的限制可能仅可以使用双卡；对于三卡或四卡工作站而言，涡轮式散热显卡或水冷散热显卡是必需，然而是否可行仍需实践。

主板与 PCI-E 总线

使用多 GPU 的前提是主板上有足够的 PCI-E x16 接口。GPU 越多，GPU 之间的数据交换就越多，对 PCI-E 通道数量的要求就越高。通常来说，有 4 块 GPU 的工作站上的每块显卡最好至少有 8 条 PCI-E 通道。这个时候 AMD 芯片组的优势就显现出来了：AMD 的新主板芯片（X570 和 X399）支持 PCI-E 4.0。就算 PCI-E 插槽只有 4 条通道，这些通道都是 PCI-E 4.0 版的，带宽相当于 8 条 PCI-E 3.0 通道的带宽。而目前 Intel 的主板只支持 PCI-E 3.0。如下图所示，长的插槽为 PCI-E x16 接口，短的插槽为 PCI-E x1 接口。

虽然有的主板上有两根以上 PCI-E x16 插槽，通常只有最上面的 PCI-E 插槽是完整的 16 条通道，第二根 PCI-E 插槽及以下的通道数量只有 4
x。这是因为 CPU 和主板芯片能够处理的 PCI-E 通道数量是有限的，除非有桥接芯片，否则 CPU 处理能力不够，需要使用主板芯片上的 PCI-E 通道。有些主板可以在插两张显卡时把一个 PCI-E 插槽的 16 条通道分成 2 组，每组 8 条通道；或者插三张显卡时把 16 条通道分成 3 组，组成 8-4-4 通道。
|CPU|支持 PCI-E 等级|通道数|
|–|–|–|
|Threadripper|4.0|64|
|Ryzen 3000/5000|4.0|24|
|10 代 Core X-Series|3.0|48|
|9 代 Core X-Series|3.0|44|
|Core|3.0|20|

由于 Ryzen 和 Core 系列仅有 24 和 20 条通道，仅能保证两张显卡的通道在 8 条；与 Threadripper 配套的 sTRX4 主板和与 Core X-Series 配套的 X299 主板上的所有 PCI-E 插槽均可以运行在 8x 以上，购买有所需 PCI-E x16 插槽数量的主板即可。

与 Ryzen 搭配的 X570 主板如果有两条 PCI-E x16 插槽，通常通道分配为 16-4（CPU-主板）；X570 主板中支持双路 8x PCI-E 的型号有：

  • ASUS AMD AM4 ROG Crosshair VIII Hero，3 条 PCI-E x16 插槽（8-8 或 8-4-4 分配）；
  • MSI MEG X570 ACE Gaming，3 条 PCI-E x16 插槽（8-8 或 8-4-4 分配）；
  • GIGABYTE X570 AORUS MASTER，3 条 PCI-E x16 插槽（8-8 或 8-4-4 分配）；
  • GIGABYTE X570 AORUS XTREME，3 条 PCI-E x16 插槽（8-8 或 8-4-4 分配）；
  • ASRock X570 TAICHI，3 条 PCI-E x16 插槽（8-8 或 8-4-4 分配）；
  • MSI MEG X570 GODLIKE，4 条 PCI-E x16 插槽（8-8-0-4 或 8-4-4-4 分配）；

与 Core 搭配的 400 系列主板只有 Z490 支持双路 8x PCI-E，其它型号的芯片只支持 16-4；貌似所有带两个 PCI-E 16x 插槽的 Z490 主板都支持双路 8x PCI-E。

对于 Threadripper 的 X399 主板来说，如果有 3 条 PCI-E 插槽，则通道数为 16-8-16；4 条 PCI-E 插槽对应的通道数为 16-8-16-8。

Core X-Seires 的 X299 主板有点复杂，因为 CPU 的 PCI-E 通道数量不同。

CPU PCI-E 插槽数 通道数
10 代 Core X-Series 3 16-8-16
10 代 Core X-Series 4 16-8-16-8
9 代 Core X-Series 3 16-8-8
9 代 Core X-Series 4 16-8-4-4

有两个型号的主板带有桥接芯片，有 7 个 PCI-E 插槽，最多支持 4 条 PCI-E x16 通道：

  • GIGABYTE X299-WU8
  • ASUS WS X299 SAGE

显卡选择

如果显卡之间有足够的空间，那么可以使用开放式散热显卡；3 块以上显卡空间有限，需要使用涡轮式散热显卡或水冷显卡。现在各个厂商只发布了开放式散热设计的显卡，下面的型号均在计划中：

  1. 涡轮式散热显卡：
  • GeForce RTX™ 3090 TURBO 24G
  1. 水冷散热显卡：
  • Colorful iGame Neptune GeForce RTX 30 系列
  • EVGA GeForce RTX 3080 10GB HYDRO COPPER
  • EVGA GeForce RTX 3090 KINGPIN Hybrid

现在是购买 RTX 30 系列显卡的好时候吗？我认为不是。

  1. 现在根本买不到啊；
  2. 各个厂家的显卡还没有开发完全；
  3. 新显卡的散热效果有待观察。

NVIDIA 已经说了，目前的缺货会延续到 2021 年。我们还是耐心等待吧。
